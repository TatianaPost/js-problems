# Сборник задач по программированию

---

## 1. Число

---

### Задача 1.1

Написать функцию `sum(x, y)`, которая:
1. принимает 2 аргумента
2. возвращает сумму своих аргументов

_Пример:_

```javascript
sum(0, 0) // 0
sum(100, 233) // 333
sum(-15, 16) // 1
```

---

### Задача 1.2

Написать функцию `diff(x, y)`, которая:
1. принимает 2 аргумента
2. возвращает разность своих аргументов

_Пример:_

```javascript
diff(0, 0) // 0
diff(100, 233) // -133
diff(-15, 16) // -31
```
